# Projeto Integrador I: Compreendendo o Negócio

### Esse projeto é uma iniciativa interdisciplinar que desafia os estudantes do 1º semestre de Ciência de Dados para Negócios da Fatec Votorantim a aplicarem seus conhecimentos em um contexto prático por meio da análise de uma empresa real de médio a grande porte.
**Empresa de Estudo**: ***Sírio-Libanês***.
